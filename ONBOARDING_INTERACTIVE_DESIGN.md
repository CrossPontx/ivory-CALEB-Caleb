## Interactive Features

### 1. Click-Through Overlay
- Overlay is **pointer-events: none**
- Only tooltip has **pointer-events: auto**
- Users can tap buttons while tooltip is visible

### 2. Auto-Positioning
- Calculates target element position using `getBoundingClientRect()`
- Adjusts tooltip position based on screen space
- Ensures tooltip is always visible

### 3. Smooth Transitions
- Tooltip moves smoothly between steps (300ms)
- Spotlight effect fades in/out (300ms)
- Pulsing ring animates continuously

---

## Animation Details

### Pulsing Ring
```css
/* Inner ring - solid pulse */
.pulse-ring {
  animation: pulse 2s ease-in-out infinite;
}

@keyframes pulse {
  0%, 100% { opacity: 1; transform: scale(1); }
  50% { opacity: 0.7; transform: scale(1.05); }
}

/* Outer ring - expanding ping */
.ping-ring {
  animation: ping 2s ease-in-out infinite;
}

@keyframes ping {
  0% { opacity: 0.75; transform: scale(1); }
  100% { opacity: 0; transform: scale(1.5); }
}
```

### Tooltip Bounce
```css
.tooltip {
  animation: bounce-subtle 2s ease-in-out infinite;
}

@keyframes bounce-subtle {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-4px); }
}
```

## Responsive Behavior

### Mobile (< 640px)
- Tooltip max-width: 280px
- Smaller text (14px)
- Positioned to avoid keyboard

### Tablet (640px - 1024px)
- Tooltip max-width: 320px
- Medium text (15px)
- More spacing around elements

### Desktop (> 1024px)
- Tooltip max-width: 360px
- Larger text (16px)
- More dramatic spotlight effect

---

## Accessibility

### Keyboard Navigation
- Tab to focus tooltip
- Enter to dismiss
- Escape to skip

### Screen Readers
- Tooltip has `role="dialog"`
